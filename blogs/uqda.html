UQDA – A Visual DSA Editor for Interactive Learning

UQDA is an interactive Data Structures & Algorithms editor that I designed and built to solve a core learning problem: static diagrams fail to explain how algorithms mutate data in real time.

React Fabric.js Rough.js Go Docker

The Problem

Traditional DSA learning tools rely on static diagrams, pre-recorded animations, or whiteboards that lack interactivity. Learners struggle to visualize how data structures mutate in real time.

The Solution

UQDA treats data structures as first-class visual entities. Arrays, trees, loops, and conditionals are interactive objects with internal state, behavior, and identity.

  • Interactive arrays, trees, and linked structures
  • Right-click context menus for operations
  • Undo/redo using Command Pattern
  • Sketch-style UI using Rough.js

Architecture Overview

The editor follows a clean MVC-inspired architecture to avoid canvas-level coupling.

CanvasManager: Handles Fabric.js lifecycle and events

Shape Classes: Encapsulate drawing logic and metadata

Zustand Store: Source of truth for shape state

Command Pattern: Add, move, update with undo/redo support

Go Backend & Real-Time Foundation

A Go backend powers authentication, room management, and WebSocket signaling for collaborative sessions.

  • JWT-based authentication
  • Room lifecycle management
  • Prepared for CRDT sync using Yjs

Containerization & Zero-Downtime Deployment

The backend is containerized using Docker and deployed using blue-green and canary strategies to ensure uninterrupted sessions.

Blue-Green Deployment: Parallel environments with instant rollback

Canary Releases: Gradual rollout for risky WebSocket changes

Health Checks: /health and /ready endpoints for traffic switching

Build → Test → Dockerize → Deploy → Health Check → Traffic Switch

Why UQDA Matters

UQDA bridges the gap between abstract algorithms and concrete understanding. By combining visual programming, clean architecture, and production-grade deployment, it moves beyond demos into a scalable educational platform.
